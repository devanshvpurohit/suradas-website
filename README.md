# Suradas — AI-Powered Wearable Assistant

Premium monochrome startup website for **Suradas** (Pre-MVP). Multipage React site with 3D hero visuals.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero with 3D scene, pre-MVP announcement |
| `/features` | Planned AI features & how it works |
| `/technology` | AI stack & terminal UI |
| `/about` | Mission, vision, testimonials |
| `/impact` | Global accessibility statistics |
| `/roadmap` | Pre-MVP → MVP → launch timeline |
| `/contact` | Waitlist signup form |

## Tech Stack

- React 19 + Vite + React Router
- Tailwind CSS v4
- Framer Motion
- React Three Fiber + Drei (3D)
- Lucide React

## Getting Started

```bash
cd suradas-website
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
